### Code example

**The task:**  
In this kata you will create a function that takes a list of non-negative integers and strings and returns a new list with the strings filtered out.  

**Solution:**  
```function filter_list(l) {
  return l.filter(function(x) {
    if (typeof(x) === 'number')
      return true;
  });
}
```
